## 🎯 What Is This?

A pass-and-play party game for **3–10 players** on a single device. Everyone gets the same secret word — except the imposter, who has to bluff their way through. Think *Spyfall* meets *Codenames*, entirely in Hebrew.

> **Perfect for:** Game nights 🎲 · Parties 🎉 · Family gatherings 👨‍👩‍👧‍👦 · Road trips 🚗

## 🕹️ How to Play

| Step | What Happens |
|:---:|---|
| 👥 | **Setup** — Add 3–10 player names and optionally enable a ⏱️ 3-minute timer |
| 🤫 | **Role Reveal** — Pass the device around. Each player privately sees the secret word — or finds out they're the imposter! |
| 💬 | **Give Clues** — Take turns giving a one-word hint about the secret word. The imposter must bluff! |
| 🗳️ | **Vote** — Everyone secretly votes for who they think is faking it |
| 🎭 | **Reveal** — Unmask the imposter, see the votes, and check the scoreboard |

🔁 Play as many rounds as you want — scores carry over!

## 📊 Scoring

| Outcome | Points |
|---|---|
| ✅ Imposter caught | Each civilian earns **1 point** |
| 😈 Imposter survives | Each imposter earns **2 points** |

## 👥 Player Count

| Players | Impostors |
|:---:|:---:|
| 3–7 | 🕵️ 1 imposter |
| 8–10 | 🕵️🕵️ 2 impostors |

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18+

### Run Locally

```bash
# Clone the repo
git clone https://github.com/romtaugranot/imposter-heb.git
cd imposter-heb

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open **http://localhost:5173** and you're in! 🎉

### 📦 Build for Production

```bash
npm run build
npm run preview
```
